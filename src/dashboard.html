Modelo

Esta API usa uma LSTM univariada treinada apenas com preços de fechamento. O treino transforma os fechamentos em log-retornos, porque variações tendem a ser mais estáveis do que o preço bruto.

No momento da inferência, o usuário envia uma lista de fechamentos. A API calcula os log-retornos, aplica o scaler salvo, executa a LSTM e reconstrói o preço previsto com last_close * exp(predicted_log_return).
