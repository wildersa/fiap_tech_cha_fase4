Arquitetura do Modelo

Esta API utiliza uma rede neural LSTM avançada, projetada para extrair padrões temporais de séries financeiras. O pipeline multivariado (V8) processa preços e injeta indicadores estacionários para maximizar a assertividade. No momento da inferência, o usuário envia os fechamentos recentes. A API normaliza os dados usando a abordagem de Anchor Price, aplica o scaler em tempo real, projeta a inferência na GPU/CPU e decodifica a previsão para o próximo preço de fechamento.
